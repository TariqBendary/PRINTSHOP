# برنت شوب - Print Shop

موقع بسيط يساعدك تطلب خدمات الطباعة أونلاين، مع حساب تكلفة الطباعة والتوصيل حسب المسافة.

---

## كيف يعمل الموقع؟

1. اختر نوع الطباعة (ملون أو أسود وأبيض).
2. أدخل عدد النسخ وعدد الأوراق.
3. اختر نوع الورق أو أدخل مقاس مخصص.
4. اكتب عنوان التوصيل.
5. الموقع سيحسب تكلفة التوصيل تلقائياً حسب المسافة.
6. يظهر لك:
   - تكلفة الطباعة
   - تكلفة التوصيل
   - الإجمالي الكلي
7. اضغط زر **إرسال عبر الواتساب** لإرسال الطلب مباشرة.
8. يدعم الموقع التبديل بين العربية والإنجليزية بضغطة زر.

---

## How the website works

- Choose print type (colored or black & white).
- Enter number of copies and pages.
- Select paper type or enter custom size.
- Enter delivery address.
- The site calculates delivery fee automatically based on distance.
- It shows:
  - Print cost
  - Delivery cost
  - Total cost
- Click **Send via WhatsApp** to place the order.
- The site supports Arabic/English switching.

---

### المتطلبات

- اتصال بالإنترنت (لحساب المسافة عبر الخرائط).
- مفتاح API من Google Maps إذا أردت استخدام خدمة Google Distance Matrix (أو يمكنك الاكتفاء بـ OpenStreetMap المجاني).

---

## طريقة التشغيل

1. افتح ملف `index.html` في المتصفح مباشرة.
2. أو ارفع الملفات إلى أي استضافة (مثل Netlify أو GitHub Pages).
